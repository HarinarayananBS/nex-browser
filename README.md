# 🚀 Nex Browser

A fast, privacy-first browser built for developers and power users.

Created by **Hari**.

---

## 🧠 What makes Nex different

Most browsers today are bloated, intrusive, and designed for mass users.

**Nex Browser** is built with a different mindset:

* Minimal resource usage
* Zero compromise on privacy
* Clean, transparent UI
* Full control for advanced users

No tracking. No clutter. No wasted resources.
